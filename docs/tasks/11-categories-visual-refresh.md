## Цель
Сделать секцию «Категории» на главной более эстетичной и премиум-ощущающейся. Текущий вариант рабочий, но сетка предсказуемая, hover банальный, фон секции — пустой.

## Контекст
- Файл: `components/Categories.tsx`. Используется только на главной (`app/page.tsx`).
- Данные: массив `categories` из `lib/products.ts` (6 элементов: krabiki, rascheski, kosmetichki, zakolki, grebeshki, sumki) с полями `slug`, `title`, `subtitle`, `description`, `image`.
- Стек: Tailwind 3.4 (палитра `rose / gold / ivory / ink`), Framer Motion 11, `next/image`, Lucide.
- Сейчас сетка: 2 колонки на mobile, 3 на lg; первая карточка раздута 2×2; фото на всю карточку + тёмный градиент снизу + subtitle капсом + title display + кнопка-стрелочка в круге.

## Решения интервью
1. **Раскладка**: журнальная асимметрия — разные размеры карточек, две акцентные крупнее остальных, "перетекание" по сетке. Не просто "первая 2×2, остальные равные", а более сложная композиция (например: 1 крупная высокая + 1 крупная широкая + 4 средних/мелких разного формата).
2. **Стиль карточки**: текст на фото + мягкий градиент (НЕ темнее текущего — теплее, с лёгким розовым тоном). Типографика крупнее и изящнее.
3. **Hover**: кинематографичный reveal — медленный zoom фото (1.0 → 1.08 за 0.8s), плавный slide подзаголовка снизу-вверх, лёгкое усиление overlay для читаемости.
4. **Декор секции**: фоновые мягкие blur-пятна в палитре бренда (например, `rose-200`/`gold-100`) за блоком, абсолютно-позиционированные с низкой прозрачностью. Опционально — тонкий орнамент (точки/линии).

## Acceptance criteria
- [x] Сетка переработана в журнальную асимметрию: на `lg` минимум 3 разных размера карточек, общее число — 6 (по числу категорий). Композиция должна выглядеть гармонично, а не как баг.
- [x] На `md` — упрощённая асимметрия (2 колонки с одной акцентной), на mobile — все одинаковые (один столбец или 2 столбца), но визуально сбалансированно.
- [x] Градиент overlay переписан: мягче, теплее (без черноты), с розовым/тёплым оттенком — не теряя читаемости белого текста.
- [x] Типографика заголовка категории — крупнее (на mobile сохранить читаемость), изящнее. Можно использовать `font-display` (Playfair) или `font-script` (Cormorant italic) для подзаголовка.
- [x] Hover: медленный zoom фото (transform-origin центр, scale 1.0→1.08, duration ~0.8s, ease-out). Подзаголовок плавно появляется снизу с opacity/translate-y. Стрелочка-кнопка реагирует элегантно.
- [x] Добавлен фоновый декор секции: 2–3 абсолютно-позиционированных blur-пятна в палитре (`bg-rose-200/40`, `bg-gold-100/30`), с `blur-3xl`, размещены за гридом, не мешают наведению.
- [x] Декор не вызывает горизонтального скролла на mobile (`overflow-hidden` на секции).
- [x] Анимации входа `framer-motion` сохранены, можно усовершенствовать (stagger, blur-in).
- [x] Сохраняется доступность: алт-тексты, focus-visible на ссылках, не теряется keyboard navigation.
- [x] `npm run build` проходит без ошибок (если запускалась задача 01 — также `npm run lint`).
