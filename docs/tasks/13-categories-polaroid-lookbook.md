## Цель
Заменить текущую типографическую ленту на живой polaroid-коллаж в духе личного дневника/лукбука: 6 фотографий-полароидов разного размера, с белыми полями и лёгким поворотом, свободно «брошенных» на страницу. Под каждым — рукописная подпись с именем категории и короткий editorial-caption. Журнальный, тёплый, тактильный.

## Контекст
- Файл: `components/Categories.tsx` (полная замена). Используется только на главной (`app/page.tsx`).
- Данные: массив `categories` из `lib/products.ts` (6 элементов: `slug`, `title`, `subtitle`, `description`, `image`).
- Стек: Tailwind 3.4 (палитра `rose / gold / ivory / ink`), Framer Motion 11, `next/image`, Lucide.
- Шрифты: `font-display` (Playfair Display), `font-sans` (Inter), `font-script` (Cormorant Garamond italic).

## Решения интервью
1. **Концепция** — Lookbook collage: разноформатные поляроиды, свободно расположенные с перекрытием, лёгкие повороты ±3-8°.
2. **Стиль фото** — классический Polaroid: белая подложка (`bg-ivory-50` или чистый белый), тонкие поля сверху/по бокам (~8-12px), толстое поле снизу (~36-48px) под подпись. Лёгкая тень. Рукописная подпись внизу в `font-script italic`.
3. **Типографика** — mix: крупные заголовки секции в Playfair (`font-display`), мелкие подписи/captions/рубрики в Inter capslocked с трекингом (`font-sans uppercase tracking-[0.3em] text-[10px]`).
4. **Детали** — caption-тексты под фото: над/под подписью добавить мелкий sans-капс caption (например, «Collection №01», «Krabiki — детали»).

## Acceptance criteria
- [x] Сетка-плитки / типографическая лента удалены. Блок построен как **свободный polaroid-коллаж**.
- [x] 6 поляроидов разного размера (минимум 3 размера: например, large ~280×340, medium ~220×270, small ~180×220). Распределение размеров — асимметричное, не повторяющееся.
- [x] Каждый поляроид:
  - белый фон (`bg-ivory-50` или чистый `bg-white`),
  - тонкие поля сверху/слева/справа (~8-12px),
  - широкое поле снизу (~36-48px) для подписи,
  - тонкая тень (например, `shadow-[0_8px_24px_-8px_rgba(184,105,115,0.18)]` — мягкая розовая),
  - лёгкий поворот (`rotate(-5deg)` … `rotate(7deg)` — индивидуально для каждого, в стиле «брошено на стол»),
  - подпись внизу: имя категории в `font-script italic` (Cormorant) среднего размера (~text-xl/text-2xl), цвет `text-ink-800` или `text-rose-800`,
  - над подписью **caption** мелким sans-капсом (`text-[10px] uppercase tracking-[0.25em] text-rose-400`) с информацией типа «Collection №01 · Krabiki» (или просто номер коллекции).
- [x] Композиция — **не строгая сетка**, а свободная компоновка с **перекрытиями** (полароиды частично налезают друг на друга на ~5-15%). Использовать CSS `transform: translate` + `z-index` для построения. Можно использовать grid с произвольными `col-start/row-start` и тонкими translate-overrides, либо абсолютное позиционирование внутри `relative` контейнера с фиксированной высотой.
- [x] Mobile (< md): композиция упрощается — повороты уменьшаются (±2°), перекрытия минимальны или их нет, поляроиды выстраиваются вертикально (один столбец) с небольшим оффсетом по горизонтали для сохранения «коллажного» ощущения.
- [x] **Hover-эффект** на полароиде:
  - лёгкое поднятие (translateY -6px, scale 1.02),
  - выравнивание поворота к 0° (или близко),
  - усиление тени,
  - z-index переходит наверх (чтобы налезающие соседи ушли под),
  - duration ~0.35s, ease-out.
- [x] Каждый полароид — `<Link href="/catalog/${slug}">`, `aria-label`, `focus-visible` ring (можно встроить в саму подпись/рамку), keyboard-фокус активирует тот же визуал что hover.
- [x] Заголовок секции (eyebrow «Категории», h2 «Шесть тёплых вселенных», параграф) — сохранить или адаптировать. Можно добавить мелкие журнальные элементы рядом с заголовком: «ISSUE 06», «p.01», тонкая горизонтальная линия.
- [x] Фоновый декор: можно сохранить мягкие blur-пятна, но добавить **бумажную текстуру** (например, через SVG-noise pattern или субтильный `bg-ivory-100` фон секции) для эффекта «страница журнала/скрэпбука».
- [x] Анимации входа: Framer Motion stagger, каждый полароид появляется с задержкой, лёгкое вращение (от случайного угла к финальному) + fade — будто их «бросают» по одному.
- [x] Доступность: `aria-label` на ссылках, декоративные элементы `aria-hidden`, keyboard nav работает.
- [x] `npm run build` проходит. Никаких новых пакетов.

## Артефакты
- `components/Categories.tsx` — полностью переписан как polaroid-lookbook коллаж. Desktop (lg+): `relative` контейнер высотой 820px, 6 полароидов в абсолютном позиционировании с перекрытиями. Три размера: large (260×220 фото), medium (210×190), small (170×150). Повороты −5°, +3°, −2°, +5°, −3°, +2°. Каждый полароид: `bg-ivory-50`, padding `px-10 pt-10 pb-42`, розовая мягкая тень, фото в `next/image fill`, под фото caption `№ XX · SLUG` мелким sans-капсом в rose-400 + имя категории в `font-script italic`. Hover/focus через `useState<number|null>`: rotate→0, y -6, scale 1.02, zIndex 30, усиленная тень. Mobile (< lg): отдельный `MobilePolaroid` компонент — вертикальный flex-col, alternating margin-left/right offset, единый размер medium, повороты ±1-2°. Stagger entry с initial rotate (увеличенный) → final rotate. priority на полароидах 0 и 3 (LCP-кандидаты). В заголовке секции добавлен журнальный акцент справа: «ISSUE 06 / p.01–06» в углу + тонкая горизонтальная линия. Под h2 — SVG-флоруш. Фон секции `bg-ivory-100 noise` (готовый класс из globals.css) + три blur-пятна (rose-200/25, gold-400/12, rose-100/20).
