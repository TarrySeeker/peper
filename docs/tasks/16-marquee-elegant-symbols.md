## Цель
Усилить бегущую ленту (Hero → Marquee) тремя точечными правками без изменения общей эстетики (font-script italic в розовом — она пользователю нравится):
1. Сделать **ярче и крупнее** (текущая `text-2xl text-rose-700/70` смотрится приглушённо).
2. Заменить разделитель `·` на **набор элегантных морских/цветочных символов** (чередование).
3. Подтвердить **бесшовное зацикливание** — текст должен повторяться без пропусков и пауз (keyframe `marquee` уже определён в `tailwind.config.ts`: `translateX(0) → translateX(-50%)`, должно работать seamless с двумя копиями).

## Контекст
- Файлы: `components/Marquee.tsx` (универсальный), `components/Hero.tsx` строки 168-179 (использование).
- Текущее состояние:
  ```tsx
  <div className="border-y border-rose-200/70 bg-white/40 py-5 backdrop-blur">
    <Marquee
      durationSeconds={45}
      itemClassName="font-script text-2xl italic text-rose-700/70"
      items={["персонализация", "подарочные боксы", "упаковка подарков", "доставка по всей России"]}
    />
  </div>
  ```
- Эстетика обёртки (`bg-white/40 backdrop-blur`, `border-rose-200/70`, `font-script italic`) — **сохраняется** (пользователь её принимает).

## Решения (без интервью — на основе сообщения пользователя)
1. **Items**: 3 пункта в формулировке пользователя — `["Персонализация", "Упаковка подарков", "Доставка"]`. С заглавных букв (выглядит читаемее в крупном размере). Пункт «Подарочные боксы» убрать — не упомянут пользователем. «Доставка по всей России» сократить до «Доставка» — короче и удобнее с разделителями.

2. **Типографика**:
   - Размер `text-3xl md:text-4xl` (вместо `text-2xl`) — крупнее, акцентнее.
   - Цвет `text-rose-700` без opacity-снижения (вместо `text-rose-700/70`) — насыщеннее.
   - `font-script` (Cormorant), `italic`, `font-medium` для усиления (опц).

3. **Разделители — чередующиеся элегантные символы**. Сейчас в Marquee жёстко вставлен `·`. Нужно вынести в props и поддержать массив-цикл. Набор:
   - `❀` — классический цветок (white florette)
   - `⚜` — fleur-de-lis (геральдическая лилия — морская/королевская)
   - `❋` — heavy chevron snowflake (флор-снежинка)
   - `☙` — reversed rotated floral heart (винтаж-флор)
   
   Чередование: после каждого пункта берётся следующий символ из массива, по кругу. Для 3 пунктов и 4 символов — каждый цикл смещение, что даёт визуальное разнообразие.
   
   Альтернативно — оставить **один и тот же** символ ❀, если чередование смотрится сложно. Решает агент после визуального осмотра — приоритет за чередованием.

4. **Цвет символов**: `text-gold-600` (#8F6A42 — насыщенное золото, контрастный акцент к rose-700 тексту).

5. **Размер символов**: `text-2xl md:text-3xl` (чуть меньше или равно основному тексту — не подавлять текст, но быть заметным).

6. **Скорость**: оставить `durationSeconds={45}` — комфортно для текущего размера. Если новый размер сильно крупнее, увеличить до 55-60.
